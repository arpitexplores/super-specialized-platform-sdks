## Why SUPER Skills For AI Vibe Coding

Super Specialized Platform SDKs helps AI coding agents work with platform-specific APIs and SDKs without losing product context. Use it for Shopify, Salesforce, Telegram bots, commerce workflows, CRM/ERP integrations, messaging systems, auth, and deployment planning.

These skills are designed for **AI vibe coding**: fast, agent-assisted building where the AI needs strong domain context, practical workflows, guardrails, and implementation-ready outputs.

**SEO and discovery keywords:** platform SDKs, Shopify development, Salesforce development, Telegram bots, commerce integrations, CRM integrations, ERP integrations, messaging bots, AI vibe coding.

## Install

Copy this folder into your agent's skills directory, then restart or reload the agent.

```bash
cp -R super-specialized-platform-sdks ~/.your-agent/skills/
```

Use it by name:

```text
Use $super-specialized-platform-sdks to help with this request.
```
